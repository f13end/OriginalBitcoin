I just posted an alpha version of F2F at
https://sourceforge.net/projects/friend2friend/. It needs PHP5, and matches
the documentation at http://wiki.friend2friend.net.
To install, unpack it and browse to the 'local users' page. Let me know how
you get on with installation if you decide to give it a try.

It still lacks some important pieces, e.g. a transport layer. However, it
the debug GUI is good enough to start learning how to program in F2F, and
the programming language is not likely to change since it has been stable
for >1 year. Thanks to Martin for helping set up the CVS.
