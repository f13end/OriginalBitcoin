This is just a reminder to the 324 participants in the Reimagining Society
Project.

(1) Please visit *
http://www.zcommunications.org/zparecon/reimaginingsociety.htm*
Check there, on the participant page, to see that you are properly listed.

(2) If you have not yet sent us a short biography, please do so by return
email.

(3) If you want to write an opening essay, or two, they are due by July 15.
If you are not listed as submitting one, but you want to, please let us know
by return email.

top page, as well.

Society site and see all the opening articles we have already posted by
using the left menu. The display and access of all Reimagining Society
content will all be dramatically improved in mid July.

Thank you...and we will try to keep email to a minimum until mid July...
